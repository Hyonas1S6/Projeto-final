    • O que é Github

    Github é uma plataforma para ferenciar seu código e criar um ambriente de colaboração entre devs, utilizando o Git como sistema de controle.

    O Github facilida o uso do Git, escondendo detalhes mais complicados de setup. É lá que você provavelmente vai ter seu repositório e usar no dia a dia.

    Possui um sistema web que permite alterar arquivos, embora isso não seja muito desejável, pois você não terá um editor, ambiente de desenvolvimento e teste.

    Para se comunicar com o Github e mexer nos arquivos do repositório, você pode usar a linha de comando, usando o comando git e suas instruções de commit, pull e push.
