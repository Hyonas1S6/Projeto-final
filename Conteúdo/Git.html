    • O que é Git

    Git é um sistema de controle de versão desenvolvido por Linus Torvalds, mesmo criador do Linux, um dos sistemas operacionais mais usados do mundo.

    Usando as ferramentas de linha de comandos que o git disponibiliza, qualquer desenvolvedor em uma equipe pode gerenciar o código fonte e seu histórico de mudanças

    Um diferencial que o Git oferece é a possibilidade de usar ramificações de recursos (feature branches) assim cada desenvolvedor pode separar uma ramificação e trabalhar em seu projeto sem fazer mudanças em seu codigo principal
